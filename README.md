# The-Riyadh-Air-Analytics-Stack
إليك وصف احترافي ومختصر جداً، يتكون من **260 حرفاً** فقط (مع المسافات) ليناسب قيود جيت هوب تماماً:  > An end-to-end BI project analyzing Riyadh Air's operational efficiency, financial profit, and cost reduction strategies.
